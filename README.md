# Último Refúgio - Sistema de Sobrevivência

# Objetivo do Sistema
Site institucional e e-commerce para a loja "Último Refúgio", especializada em equipamentos de sobrevivência.
O projeto visa aplicar conceitos fundamentais de HTML e estruturação web.

# Tecnologias Utilizadas
* HTML: Estruturação semântica de todas as páginas.
* CSS: Estilização inline para garantir a identidade tática (fundo escuro e tipografia legível).
* GitHub Pages: Para hospedagem e visualização online.

# Estrutura do Projeto
* `index.html`: Página inicial com destaques e notícias.
* `produtos.html`: Catálogo de equipamentos.
* `noticia.html`: Detalhes sobre o Workshop na Selva.
* `carrinho.html`: Simulação de inventário (tabela de compras).
* `contato.html`: Formulário de comunicação.

# Como executar o projeto
1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
